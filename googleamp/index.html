Topic: Were Cats Radder In The 90's?

By FAKE FAKERSON

Although Googles AMP Project is one of the newest and most interesting technologies to hit the mobile web, it is in a lot of ways a throw back to 90's web design. Similar to the resurgence of table based design in modern HTML email scaffolding, practices once (and still) frowned upon or forgotten can be currently seen at the top of googles mobile search site via the news carousel. Things like explicitly defined image sizes, inline styles and style tags in the head of the page have been abandoned for more responsive elements and modular maintainable code but have been reborn as part of this new specifications designed for mobile speed.
